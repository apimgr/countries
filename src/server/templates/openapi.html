Countries API

GET /api/v1/countries

Get all countries from the database.

GET /api/v1/countries/search?q={query}

Search countries by name with case-insensitive matching.

Query Parameters

Name Type Description
q required string Search query (min 2 characters)
GET /api/v1/countries/{code}

Get a country by ISO 3166-1 alpha-2 code.

Path Parameters

Name Type Description
code required string Country code (e.g., US, GB, JP)

Geolocation API

GET /api/v1/coordinates?lat={lat}&lon={lon}

Find the nearest country to given coordinates.

Query Parameters

Name Type Description
lat required number Latitude (-90 to 90)
lon required number Longitude (-180 to 180)
POST /api/v1/coordinates

Find the nearest country (JSON body).

Request body: {"lat": float, "lon": float}

GET /api/v1/nearby?lat={lat}&lon={lon}&radius={km}

Find countries within a radius of given coordinates.

Query Parameters

Name Type Default Description
lat required number - Latitude (-90 to 90)
lon required number - Longitude (-180 to 180)
radius number 500 Search radius in km

Statistics & Utility

GET /api/v1/stats

Get API statistics including country count.

Also available as /api/v1/stats.txt for plain text format.

GET /api/v1/count

Get total country count.

Also available as /api/v1/count.txt for plain text format.

GET /random

Get a random country.

Also available as /random.txt for plain text format.

GET /data

Get the raw country database as JSON.
